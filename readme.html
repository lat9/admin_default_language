What it does

You've got a multi-language store and you'd like the default language used in your Zen Cart admin to be different than the default language used in your store. This plugin defaults your admin-panel's language to English, but you can change the behavior to use any of your installed languages. Just change one line in the file provided by this plugin, changing the en to one of the 2-character language codes supported by your store:

if (!defined ('DEFAULT_LANGUAGE_ADMIN')) define ('DEFAULT_LANGUAGE_ADMIN', 'en');  //-Default admin to English

If you are logged into your admin when you make this change, you'll need to log out and then back in to have the change take effect!

Installation

There is one core-file overwrite in this plugin; you should always backup your cart’s database and files prior to making any changes.

Copy the file to your store's file-system, after renaming the "YOUR_ADMIN" directory to match your custom admin folder's name:

  1. /YOUR_ADMIN/includes/init_includes/init_languages.php
